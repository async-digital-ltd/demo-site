# demo-site

Live product demos from Async Digital, hosted at [demo.async-digital.com](https://demo.async-digital.com) via GitHub Pages.

## Structure

- `index.html` — landing page listing available demos
- `CNAME` — custom domain config for GitHub Pages
- `speed-to-lead/` — missed-call follow-up demo

## Adding a demo

1. Create a new folder at the repo root (e.g. `my-demo/`) containing an `index.html`.
2. Add a link to it from `index.html`.
3. Commit to `main` — GitHub Pages deploys automatically.
